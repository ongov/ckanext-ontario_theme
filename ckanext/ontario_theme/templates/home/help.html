    {% trans %}Q. I cannot find the dataset I’m looking for.{% endtrans %}

    {% trans %}A. Try using the catalogue search page. Enter your keywords in the search bar and refine your search, using the filters provided.{% endtrans %}

    {% trans %}Q. How do I download datasets?{% endtrans %}

    {% trans %}A. On each page in the catalogue, where data is available, you will find the data icon under the heading Data and Resources.{% endtrans %}

    {% trans %}To the right of the data icon, you will find a either a Download or Open button.{% endtrans %}

    {% trans %}Click the Download button and the data will be downloaded immediately to your computer.{% endtrans %}

    {% trans %}Click the Open button to find out how to access the dataset (which may be hosted outside the catalogue).{% endtrans %}

    {% trans %}Q. What does the Preview button do?{% endtrans %}

    {% trans %}A. The Preview option lets you see a dataset without downloading it. You will see this option for files that are stored as common file types, like CSVs.{% endtrans %}

    {% trans %}When you click the Preview button, you will get to a page that:{% endtrans %}

    • {% trans %}describes the files (including a description of the file and the web address where the file is stored){% endtrans %}
    • {% trans %}allows you to see the data in a few different ways - in a spreadsheet, as a graph or geographic map{% endtrans %}

    {% trans %}The Graph and Map options are available just above the dataset.{% endtrans %}

    {% trans %}Q. How does the Graph option work?{% endtrans %}

    {% trans %}A. When you click the Graph button, you will get to a screen that allows you to set the parameters of the graph you want to see.{% endtrans %}

    {% trans %}Once you select your graph type, and your axes, you should see a simple graph:{% endtrans %}

    {% trans %}Q. How does the Map option work?{% endtrans %}

    {% trans %}A. When you click the Map button, you will get to a screen that allows you to identify the mappable information (latitude and longitude) in the file. You should then see a simple map:{% endtrans %}

    {% trans %}Q. What if I don’t see a Preview button?{% endtrans %}

    {% trans %}A. If there is no Preview button for a dataset, then you will see an About button. This means that the dataset contents cannot be previewed as a spreadsheet, graph or map by the catalogue.{% endtrans %}

    {% trans %}When you click About, you will get a description of the file.{% endtrans %}

    {% trans %}Q. What if a dataset isn’t available in the format I need?{% endtrans %}

    {% trans %}A. For any issues related to our datasets, email us at opengov@ontario.ca or send us a message through our contact us form. We will review your issue and reply.{% endtrans %}

    {% trans %}Q. How do I request an updated version of a dataset?{% endtrans %}

    {% trans %}A. Start by checking the Update frequency field to see how often the dataset should be updated.{% endtrans %}

    {% trans %}If the dataset hasn’t been updated as frequently as stated, send an email to the dataset maintainer, using the Maintainer Name and Maintainer Email fields.{% endtrans %}

    {% trans %}If a maintainer isn’t listed, email us at opengov@ontario.ca or send us a message through our contact us form.{% endtrans %}

    {% trans %}Q. How do I report errors/issues/provide feedback (for example, inaccurate, outdated data)?{% endtrans %}

    {% trans %}A. Start by sending an email to the dataset maintainer, using the Maintainer Name and Maintainer Email fields.{% endtrans %}

    {% trans %}If a maintainer isn’t listed, email us at opengov@ontario.ca or send us a message through our contact us form. Please specify which dataset(s) you’d like us to review.{% endtrans %}

    {% trans %}Q. Why isn't the data for a dataset available?{% endtrans %}

    {% trans %}A. There are three reasons why the data for a dataset might not be available:{% endtrans %}

    • {% trans %}The dataset is To be opened (you will see this on the catalogue page). The government has screened the files for any privacy concerns and is prepping them to share with the public.{% endtrans %}
    • {% trans %}The dataset is Under review. The government has not finished screening the files. The data will not be available in the short-term and may never actually be published in the case that screening determines it isn’t safe to share with the public. You can check back to see if the status changes.{% endtrans %}
    • {% trans %}The dataset is Restricted. The government has screened the files and determined that they are not safe to share with the public.{% endtrans %}

    {% trans %}Q. Can you notify me when the catalogue is updated?{% endtrans %}

    {% trans %}A. Yes, you can get data catalogue updates by RSS feed. RSS (Really Simple Syndication) lets you know when our data catalogue has been updated.{% endtrans %}

    {% trans %}To receive RSS updates:{% endtrans %}

    1. {% trans %}Install an RSS reader on your device{% endtrans %}
    2. {% trans %}Choose the updates you want to receive{% endtrans %}

    {% trans %}Available updates:{% endtrans %}

    • {% trans %}Any dataset is updated or added to the catalogue{% endtrans %}
      RSS URL: data.ontario.ca/feeds/dataset.atom
    • {% trans %}A specific dataset is updated{% endtrans %}
      RSS URL: data.ontario.ca/feeds/dataset.atom?id=[{% trans %}name-of-specific-dataset{% endtrans %}]
      {% trans %}Example{% endtrans %}: data.ontario.ca/feeds/dataset.atom?id=zooplankton-monitoring
    • {% trans %}A dataset with a specific tag is updated or added{% endtrans %}
      RSS URL: data.ontario.ca/feeds/tag/[{% trans %}name-of-tag{% endtrans %}].atom
      {% trans %}Example{% endtrans %}: data.ontario.ca/feeds/tag/pepper.atom
    • {% trans %}A dataset based on your custom search term is updated or added{% endtrans %}
      RSS URL: data.ontario.ca/feeds/custom.atom?q=[{% trans %}custom search term{% endtrans %}]
      {% trans %}Example{% endtrans %}: data.ontario.ca/feeds/custom.atom?q=apple
    • {% trans %}A specific ministry updates or adds a dataset{% endtrans %}
      RSS URL: data.ontario.ca/feeds/organization/[{% trans %}name-of-specific-ministry{% endtrans %}].atom
      {% trans %}Example{% endtrans %}: data.ontario.ca/feeds/organization/agriculture-food-and-rural-affairs.atom
